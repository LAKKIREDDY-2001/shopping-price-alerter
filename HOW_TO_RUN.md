# How to Run

1.  **Install dependencies:**
    ```bash
    pip install Flask Flask-Cors requests beautifulsoup4 Werkzeug
    ```
2.  **Run the Flask application:**
    ```bash
    python app.py
    ```
3.  **Access the application:**
    Open your browser and navigate to `http://127.0.0.1:5000`. You will be redirected to the signup page. After signing up, you can log in and access the dashboard.
