# AI Price Tracker - Browser Extension

A powerful browser extension that tracks prices on Amazon, Flipkart, Myntra, and other shopping websites. Get instant notifications when prices drop!

## Features

- One-click Price Tracking
- Instant Notifications
- Price History & Trends
- Auto Sync with Dashboard
- Dark Mode Support

## Supported Sites

- Amazon (IN, COM)
- Flipkart
- Myntra
- Ajio
- Meesho
- Snapdeal
- Tata CliQ
- eBay

## Installation

### Chrome/Edge/Brave
1. Open `chrome://extensions/`
2. Enable Developer mode
3. Click "Load unpacked"
4. Select the browser-extension folder

### Firefox
1. Open `about:debugging`
2. Click "This Firefox"
3. Click "Load Temporary Add-on"
4. Select any file in browser-extension

## Files

- manifest.json - Extension config
- popup.html/css/js - Extension popup
- content.js/css - Site scripts
- background.js - Service worker
