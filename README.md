# Price Alert - GitHub Pages

A beautiful, responsive web application for tracking price drops and alerts.

## Pages

- **Sign Up** (`signup.html`) - Create a new account
- **Sign In** (`signin.html`) - Login to your account
- **Dashboard** (`dashboard.html`) - View your price alerts and statistics

## How to Use

1. Open `signup.html` to create an account
2. Sign in with your credentials
3. View your dashboard with price alerts
4. Navigate through the sidebar to access different features

## Deployment

This project is designed for GitHub Pages deployment:

1. Create a new repository on GitHub
2. Push these files to the repository
3. Go to Settings > Pages
4. Select the main branch as source
5. Your site will be live at `https://yourusername.github.io/repo-name/`
